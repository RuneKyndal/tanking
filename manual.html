Tanking App manual. this is a very early release, all suggestions are welcome. the source code is published here, for review or if you want to fork it and build your own from here. https://github.com/RuneKyndal/tanking/ quick start. the app is designed as a "WebApp" for it to work properly when offline. it must be installed. not just a saved shortcut. IOS use Safari browser (not chrome) go to link. once loaded "share" + more share to home screen ensure "web app" tickbox id ON close everything. and launch from homescreen to do initial download. ensure to give all permissions to app if asked. Android. suggested. Chrome browser go to link. once loaded click "add to home screen" click the Install NOT shortcut option. close browser, and launch from homescreen to do initial download. ensure to give all permissions to app if asked. Start choose aircraft type, update weights and fuel load. [go start] if desired, use engine timers [go fly] (starts flight timer) in flight ensure GPS activates verify fuel burn rate, and desired target ground speed. occasionally update fuel load to match aircraft as fuel is burned. flight log click the [back] button to return to timer page, stop/start timers, save to log, export logs
